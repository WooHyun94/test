1. 현재 홈페이지 상태


현재 상태는 기본적인 UI를 바탕으로 상단 메뉴바 Home, Assignment, Report 버튼에 각각 html을 연결해 둔 상태이며 CSS 파일을 따로 만들어 모든 html 파일에 연결해둔 상태입니다. UI는 제가 자주 이용하는 사이트인 로아와라는 사이트처럼 접속한 디바이스에 상관 없이 동일한 모습으로 보이게끔 하고 싶었고 구현할 방법을 찾다 부트스트랩 을 사용하게 되었습니다.


1.1 Home


index.html 파일에 구현해둔 화면으로 각 과목별로 ass_과목.html 파일을 연결해 두었고 시간을 체크해서 수업시간 한시간 전부터 시작할 때까지는 해당 수업 태두리를 주황색으로, 해당 수업의 수업시간에는 빨간색으로 표시되게 했고 정적 사이트이기에 실시간으로 시간을 확인할 수 없어 화면 우측하단 시간체크 버튼(->index.html)을 통해 시간을 체크할 수 있게끔 만들었습니다.


1.2 Assignment
